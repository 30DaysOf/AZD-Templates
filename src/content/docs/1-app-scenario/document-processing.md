# Intelligent Document Processing

Learn how to build intelligent document processing solutions using Azure AI services and AZD templates.

## Overview

Intelligent Document Processing (IDP) combines OCR, natural language processing, and machine learning to extract, classify, and process information from documents automatically.

## Key Components

- **Azure Form Recognizer**: Extract text, key-value pairs, and tables from documents
- **Azure Cognitive Services**: Process and understand document content
- **Azure Functions**: Orchestrate document processing workflows
- **Azure Storage**: Store and manage documents securely

## Use Cases

- Invoice processing and validation
- Contract analysis and extraction
- Healthcare records digitization
- Legal document review

## Getting Started

Use the AZD templates to quickly deploy an IDP solution with pre-configured Azure resources and sample code.
