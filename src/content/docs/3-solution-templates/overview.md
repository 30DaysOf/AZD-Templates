# Solution Templates Overview

Explore our collection of ready-to-use templates to accelerate your AI solution development.

## Available Templates

Our templates cover various AI scenarios and are production-ready with:

- **Infrastructure as Code**: Bicep templates for Azure resources
- **Application Code**: Sample implementations in multiple languages
- **CI/CD Pipelines**: GitHub Actions and Azure DevOps workflows
- **Documentation**: Comprehensive setup and usage guides

## Template Categories

- **Chatbot & Conversational AI**: OpenAI-powered chat applications
- **Document Intelligence**: Document processing and analysis
- **Search & Retrieval**: RAG-based search solutions
- **Data & Analytics**: AI-powered data processing
- **Computer Vision**: Image and video analysis solutions

## Using Templates

All templates are designed to work with Azure Developer CLI (azd):

```bash
azd init -t <template-name>
azd up
```
