### Prerequisites

- Node.js 20 or higher
- npm or yarn
- (Optional) Python 3.12 for Python-based examples

### Local Development

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The site will be available at `http://localhost:4321`

### Using Dev Container

This project includes a `.devcontainer` configuration with Node.js 20 and Python 3.12 support. Open in VS Code and select "Reopen in Container" when prompted.
